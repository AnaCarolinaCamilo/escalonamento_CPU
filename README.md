# escalonamento_CPU
## Trabalho 1 para a matéria Sistemas Operacionais I
#### O trabalho consiste em mplementar um conjunto de algoritmos de escalonamento de CPU e escrever um programa que calcula uma série de estatísticasbaseado nestes algoritmos. Esses sendo:
* FCFS: First-Come, First-Served
* SJF: Shortest Job First
*  RR: Round Robin (com quantum = 2)
